# 以太坊当前值多少钱

- [ ] 以太坊估值及安全边际 #active 

ETH 的估值不能简单用网络燃烧作为收入的逻辑，因为这把它看成了股票，这个角度看它可能非常高估。股票的价值捕获主要是企业的未来现金流的拥有权，而ETH 是完全不同的逻辑。

ETH = Commodity（商品） + Monetary（货币） + Security（安全资产）

商品角度，它代表了对去中心化区块和计算的需求。从这个角度看，什么样的情况会产生对它的巨大需求？
- 高价值结算（High-value settlement）
- 金融原生交易密集期（DeFi / 杠杆周期），估值类似于交易所
- 不可替代的去中心化需求，去政府化
- 系统性迁移（主权级别，最强但最难发生）

目前正在发生的：RWA与稳定币
主流金融系统结算迁移尚未开始

还有一些宏大叙事，比如对现有金融体系的灾备，成为大机构的最终账本，single source of truth，人类多星账本（降低一层，多主权世界，跨 AI 和人类，全球资产）

约束：隐私和监管，性能，可能终局不是所有人都直接用 ETH 记账，而是ETH 提供最终结算与可验证性

Monetary 指可以作为金融系统里的工具，如抵押品，记账单位，它也可以作为作为一种储值工具（sov）被储备。这个语义下最具爆发力的场景不是在正常市场环境，而是在法币失信的极端情境下。当某个经济体的人需要把财富转移到政府够不到的地方，ETH 的货币属性才会被真实需求激活。当然它也有机会和现有的体系融为一体，越来越多经济活动逐渐迁移到 ETH。
现状：在 Defi 已经部分实现，但是 Defi 还不够大。对于 sov，人们更倾向于 BTC，因为它的叙事更简单且有总量上线。ETH 的 monetary 属性只有当它的网络里有足够多的经济活动时才能体现。

如果 Commodity 决定需求，Monetary 决定上限，那么 Security 决定“是否存在”以及“底部在哪里”。在以太坊中，ETH 还承担着维护系统安全的责任。
当网络价值上升时，网络就需要更安全，依赖更多 ETH 被质押。质押意味着减少 ETH 的流通量。
安全预算：ETH Price × Staked ETH ≈ Security Budget
提升安全预算，要么提升质押量，要么提升 ETH 价格。安全预算可以为 ETH 提供价格的下限，哪怕 defi 活动减少，只要网络上要资产，就需要最低的安全预算。
从这个角度来说它也提供一种期权价值，是以太坊网络未来价值的期权。如果以太坊真的成为全球金融基础设施的一部分，现在持有 ETH 就是在极早期买入这个网络的"股权"。这个期权的行权条件就是上面梳理的两层：Commodity 需求爆发 + Monetary 临界点到达。

ETH 的价值还有人的影响，因为以太坊本质上是一个软件平台，就想之前 L2 的战略就削弱了它的价值捕获能力，而最近又重新聚焦 L1 又让它变好了。这个倒是类似股票，有企业管理层的影响，虽然它是去中心化的，但它仍然有一个基金会及核心客户端团队可以一定程度主导它的方向。

类似的公司的估值以及比较角度
从 Commodity 度看，它有点像 AWS 或者微软，贩卖去中心化存储及计算的使用权，但是它不像这些企业有定价权，不能稳定的抽成。AWS 的 PS 与 ETH 对比可能会得到比较低的估值。
Monetary 的角度，它对应黄金或国债。这部分看规模和地位，本身也是自我强化的。
从 security 看，它有点像是金融基础设施，类比于交易所活清算机构，比如纽交所或 CME。看网络期权价值有点像早期的 visa。这几个再分别类比做一些估值。
