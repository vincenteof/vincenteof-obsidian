# Valuation for CRCL

- [ ] learn the basic of CRCL #active 

分析 CRCL 首先要分析稳定币这个生意。它和保险公司有点像，保险通过用用户的浮存金进行投资，而稳定币发行公司通过用户储备的资产买入国债。从这个角度而言，稳定币发行公司甚至会比保险公司更好，因为铸造出来的稳定币本质上是零利息负债。
从另一个角度看它又是一个卖铲子的公司。用户愿意放弃真实世界的利息选择稳定币，本质上是预期通过稳定币获取更多的收益。目前可见的是大部分稳定的使用场景都在币圈，持有稳定币就是币圈的入场券。因此这个生意必然也具有周期性，和币圈本身的周期性相当一致。当币圈进入牛市，对稳定币的需求也必然旺盛。
币圈本身高利润来源类似于是赌博需求，这个方向已经比较成熟了，比如参与各种山寨币和合约。而另一个方向是 Defi，它没有具体的形式，终极目标是要将所有金融活动重塑，因为上升空间极大，比如各种金融资产的代币化，而此类活动几乎都需要稳定币。
另一个方向可能是跨境的资金转移，这个方向并没有那么性感，可能会降维打击一些做跨境支付的公司及地下钱庄。目前这种需求可能也已经比较成熟，需要扩大范围还是需要更多监管相关的政策。
还有一个点是美债，美国政府长期需要解决美债派发的问题，稳定币一定程度上可能可以有一定作用。
因此，从需求端看未来稳定币的增长空间是很大的，它能捕获的价值理论上不会比 ETH 这种公链少。


除了体量规模外，另一个问题就是盈利能力。目前最大的问题就是盈利单一化，几乎完全依赖美联储的利率。这个问题可以一定程度被规模的扩展抵消，但是利率周期性影响不可忽视。目前的监管框架下，稳定币不能给持有人利息，这是利好稳定币发行商的。如果后续监管改变，需要让持有者得到利息，利润率还会下降。

稳定币的护城河在于：
1. 合规。因为规模的扩大几乎都依赖传统金融的入场，因此合规的重要性非常重要。
2. 网络效应。
