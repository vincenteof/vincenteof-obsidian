# 如何走出牛市

美股一般 5 年一次小危机，10 年一次大危机。虽然美股熊短牛长，但是一旦熊市降临，会猛烈快速地下跌，对于指数而言一般有30%的幅度，对于个股可能超过 50%。因此哪怕你是高抛低吸，但是从逻辑上来说，总会有一次抄底消耗完你投资账户里的所有现金，并且市场依然没有到底。
因此，走出牛市比牛市本身更为重要。
对于类似我这样的零售投资者，完美择时去空仓甚至做空是超出能力范围的任务。我的思路是建立一套现金流永远不会断裂的机制去度过熊市，去购买别人被迫卖出的资产。

## 第一条：把收益"锁定"

它大概是这样：
```
工作收入      
	│      
	▼
成长账户（Alpha）TSLA / NVDA / AI Infra ...      
	│      
	│ 牛市获得超额收益      
	▼
定期收割一部分利润      
	│      
	▼
退休账户（Beta）VOO / BRK.B / SCHD / 短债      
	│      
	▼
持续产生：
	- 股息
	- 利息
	- 更低波动
```

整个系统形成一个闭环。

但对于已经获得巨大浮盈的人来说，与其 never sell 不如 **把一部分纸面收益，永久转换成未来的现金流。** 而且对心态也能产生积极的影响。你的浮盈以后可能会每月稳定地给你产生利息，这些现金流可以在熊市继续流入成长账户。

注意，**不是每年转钱，而是只在"超额收益年"转钱。**
我个人对于成长账户的要求，超过 20% 我就认为是超额收益了。

而且对于成长账户而言，基准是 QQQ，在每年没有结束的时候，如果浮盈巨大也可以 rebalance 到 QQQ，这样也能更好地保护自己的收益。QQQ 就是你的机会成本，永远都是和自己的机会成本比，如果有更好的机会，就从 QQQ 切换到更高 beta 的标的。

## 第二条：保持场外收入

因为对于一个还在积累财富的人来说：
**工资，其实就是最大的现金流资产。**
每月到账的工资可以让你的 DCA 永远不会停止。
