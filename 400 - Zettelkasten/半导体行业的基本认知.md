# 半导体行业的基本认知

NVDA 生产一块 H100 的过程：
1. NVDA 只做芯片设计及软件生态，他们讲芯片蓝图交给 TSMC 进行制造。
2. 制造先从硅开始，高纯度硅会被做成单晶硅棒，再切成晶圆（Wafer）。整个 GPU 芯片，会在这张圆盘上批量制造。
3. 通过光刻把电路印到硅上，需要 ASML 的 EUV 光刻机。
4. 通过刻蚀把多余的部分挖掉，因为芯片不是二维结构，而是多层的，因此这些步骤会不断重复。
5. 芯片需要导电层，绝缘层，金属连接层。沉积设备会像：“原子级喷涂机”。把极薄材料铺到晶圆上。
6. 检测与良率，因为一个晶元上只要一部份的 CPU Die 能正常工作。
7. 通过切割切成一块块 CPU Die，这时这时它还不是显卡。只是：“裸芯片”。
8. 通过 TSMC 的 CoWoS 把 GPU Die，HBM，高速连接层。
9. 之后 Nvidia 或 ODM 厂商会：加 PCB，加供电系统，加散热，加网络模块。最后变成显卡。

Nvidia 负责设计“超级大脑”，  
TSMC 用世界最先进工艺把它刻进硅里，  
ASML 用 EUV 把纳米级图案投影出来，  
Zeiss 负责控制光，  
韩国公司提供超高速记忆系统，  
最终全球数据中心把这些芯片变成 AI。

大部分组装加工过程发生在 TSMC，里面整合了各种需要的设备，比如光刻机和各种纯材料。它最大的护城河在于它的良率，普遍认为它先进制程的良率能达到 70% - 90%，而且在开辟新制程的时候，良率爬坡速度极快。在半导体制造行业，良率几乎决定一切。（竞争对手：三星，INTC，中芯国际）

先进制程高度依赖 EUV，成熟制程主要是用 DUV。DUV 也可以多次曝光做先进制程，但是进入 7nm 以内成本爆炸。而且先进制程收益递减已经逐步发生，比如2nm 的成本暴涨，但性能提升不再像过去那样指数级。于是与其把单颗芯片无限缩小，把多颗芯片高速连接。这也是 CoWoS、Chiplet、HBM 越来越重要的原因。

本质上来说最后都会通向物理学：
AI → Nvidia → TSMC → ASML → Zeiss → Optics → Physics


整个流程中大部分公司都是工业垄断型公司，NVDA 是平台型公司。平台最大的风险是范式切换；而工业垄断最大的风险则更多来自周期、地缘政治和技术替代。
