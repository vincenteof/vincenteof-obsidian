# Stoicism

- [x] learn the basic ✅ 2025-12-11
## Basic

什么是斯多葛主义：**如何在一个不可控的世界里，保持内在的自由和行动力**

>可控的：你的行为、判断、意志。  
>不可控的：市场走势、他人评价、经济周期、运气、情绪波动本身。

策略：**把全部精力投注在自己的可控变量上。**
斯多葛主义给你的是：
- 重新夺回对注意力的控制
- 不被外部波动牵着走
- 在长期里让优势复利

斯多葛主义围绕三件事：认知，行动，内在平静
**认知（Judgment）——看清外部世界是什么**
大部分痛苦来自把“事件”解释成“威胁”。
- 市场下跌不是灾难，是噪音
- 别人的评价不是命令，是风声
- 当下的情绪不是你，是天气
斯多葛训练你把“事件”和“解读”剥离，让你能获得真正的清晰。
**行动（Action）——把焦点放在能提升长期曲线的事**
斯多葛不是被动忍耐，它强调：
- 纪律
- 行动
- 勇气
- 反脆弱的选择
长期主义者所有的优势都来自——  在别人恐慌时你能保持冷静，在别人兴奋时你又能保持自律。
**内在平静（Will）——和不可控的世界和解**
你不能阻止挫折，但可以决定面对它的方式。
斯多葛主义强调：
- **自我主权（Self-Ownership）**
- **抗压能力（Resilience）**
- **情绪容纳，而非压抑**
斯多葛主义本质上就是训练你的前额叶在关键时刻“重新上线”

斯多葛主义者对幸福的定义：
> 幸福 = 内在的安宁（Eudaimonia），来自于活得与自己的理性、价值、与德性一致。

换句话说，  
**幸福不是情绪，而是状态；  不是高潮，而是“内心秩序井然”的感觉。**
这在神经科学里就是：  
**前额叶（理性系统）成功管理边缘系统（情绪系统），让人进入一种平稳、主权、自洽的状态。**

斯多葛主义不否认外界有快乐、财富、成功、荣誉，但认为：
- 这些东西都是 **“preferred indifferent”（可取但不依附）**
- 它们能带来快感，但 **快感不是幸福**
原因很简单：
**外在的东西不稳定，因此依赖它们获得幸福，会制造永久不稳定。**

斯多葛主义者追求的是：
> 只在可控事物上构建人生的稳定性。

斯多葛主义认为快乐分两类：被动快乐（pleasure）与 主动幸福（eudaimonia）
前者来自获得外物：涨幅、赞美、好运、目标达成、消费、娱乐  
	→ 快速上升、快速回落、强依赖外界  
后者来自于：行为与价值一致，自己尊重自己，内心稳定、有序、掌控  
    → 持久、可复利、不依赖外界

斯多葛主义者定义的不幸：
> 当你知道正确的道路，却被自己的情绪、欲望与冲动拖走。

这其实就是“前额叶失效、边缘系统接管人生”。

## Train

 控制圈练习 －不在可控的事件上投入精力
 负面可视化－给自己合理的心#理预期
 情绪分离－深呼吸，把更高的自己显现，让掌管自己的身体
 自愿的不适－可以选择一些不舒服的小事
