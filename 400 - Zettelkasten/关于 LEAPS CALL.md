# 关于 LEAPS CALL

- [x] 总结 LEAPS CALL 相关基本认知 ✅ 2026-06-16

LEAP call 本质上也是杠杠，是带时间期限不会爆仓的杠杆。
有些机构喜欢攻 LEAP 代替正股获得股票的 exposure：
100% SPY -> 20% SPY LEAP CALL + 80% 国债

这种方案的核心思想是资本效率，获得同样的敞口，但是剩余的资金还可以用来赚钱。
这种方案一般会选择偏实值的 CALL，在接近到期如果还亏损那么就 roll 或者转成现货。

这张方式还是心智包袱太重，总收益只能多几个百分点，对个人投资者意义不大。
我认为它有用的是作为暴跌后的进攻工具。比如某年市场大跌，SPY 超过 20% 的跌幅。
