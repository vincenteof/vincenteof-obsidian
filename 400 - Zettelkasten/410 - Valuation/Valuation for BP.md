# Valuation for BP

## 代币经济学

### 分配
BP 代币 25% 会在 TGE 完全分配给社区，37.5% 在 Pre-IPO 根据里程碑事件解锁给用户，剩下 37.5% 直接作为公司的储备资产。
币给内部员工带来的激励可能有两种模式：
1. BP 代币可以看作一种可转债，等 IPO 以后可以转股
2. 公司作为 BP 代币的 DAT，代币价值提升将带来股票的估值上升
具体要看后期在当前监管下能做到什么地步。
https://x.com/armaniferrante/status/2020863914918158535
https://x.com/Metabape/status/2021132109323305237

BP 的 token 在锁仓一年后可以置换为股权，比如为总股权的 20%。目前已知的总流通为 25% TGE + 37.5% 的合规/增长释放，总计 62.5%。如果全部质押一年的花，62.5 / 20 = 3.125，兑换比例为 1:3，如果 1/3 的人质押一年，那么兑换比例为 1: 1。目前 BP 股权的估值为 1B，而代币估值只有 500M。
