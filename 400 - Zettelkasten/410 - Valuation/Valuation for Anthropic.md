# Valuation for Anthropic

美股 7 巨头对 Anthropic 持有合集 38.5% 的股权，其中亚马逊占比最大，约 20%。

因为 Anthropic 专供高 token 消耗的 2B 行业，因为盈亏控制的比 OpenAI 好。还有一种负面观点是，当年uber facebook amazon那类公司，巨额亏损的时候都是处于垄断地位。而这些AI公司还没垄断亏损就远超那些巨头最亏的年份了。

2026/2/28 OAI 升级到了 Codex5.3，在保证模型质量的同时提升了速度，可以开始与 CC 在 agent 和编程分庭抗礼，OAI 可以发挥其大用户基数优势。大模型之争还没结束。
