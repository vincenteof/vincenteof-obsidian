# 中国房地产的终局

大量二手房供应，包括银行直供房。

## 估值
一种算法是收益率和国债/美债对比。这种方式最大的问题是需要估算很多其他因素：
1. 核心地区的房产不止有收入属性，也具有一定的安全/保值属性
2. 国外房产的持有成本比国内高
3. 国债本身是保本的，房子本身具有波动性，这个点会在房产的上行和下行区间产生不同的影响
4. 人民币无法自由流通
