# 人生复利思维

## insight
选择长期有复利效应的事，这样时间就成了你的资产而非成本。复利性思维意味着任何事情在不需要马上回报、甚至短期不太“划算”的事情里持续投入，因为你知道时间会把回报从线性变成指数。
有时候短期越不想做的事越要去做，回过头去看都是积累复利的最佳时机。

慢慢变富的方法实操上更具有可行性，一是不需要危害自己健康去追逐虚幻的泡沫，二是日常精神压力和时间压力小很多，三是因为克制的生活方式可以健康的活很多年，才能真正享受财富的长期复合增长。那些因为着急发财，不到六十岁就突然去世的人，其实还是没有把这个道理彻底想清楚。Horejsi 开始把日常生活的基调定为游手好闲但稳健的复合增长，而 Semel 六十多岁还要忙忙碌碌为人打工不断奔波于各种复杂交易和人事关系时，两个人未来的不同命运早已注定。

## action
- 每周足量运动
- 睡眠时间充足
- 保持神经系统平静
- 远离让自己觉得不舒服的事物
- 保持学习与思考
- 皮肤护理
- 珍惜注意力
