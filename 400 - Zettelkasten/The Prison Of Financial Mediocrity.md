# The Prison Of Financial Mediocrity

- [x] 完成平庸之金的牢笼的总结及具体操作 #active ✅ 2026-01-24

核心观念就是：**Long Degeneracy**

传统财富积累之路已然封闭，婴儿潮以 20%人口占比坐拥近半数国家财富，而占据同等人口比例的千禧世代却只握有 10%财富。资产价格通胀恰巧惠及了那些早已拥有资产的人

基本的温饱问题已被解决，年轻人会追求下一个层次：归属感、尊重和自我实现。这些都需要经济上的巨大成功，通向这些更高需求的传统途径已经被堵死了。大多数行业没有巨大的需求和前景，白领阶层被 AI 迅速挤压，社交媒体让人永远对自己所处的位置不满意。

他们看不到能通向他们梦想的路径，在这种情况下，年轻人更倾向选择一种理性的非理性，趁着资金和机会尚存，现在放手一搏。

不论是加密货币，NFT，金融衍生品或者预测市场，都是这种巨大需求的出口。

如果我们不想参与这种赌徒游戏，我们应该选择什么？简单来说我们需要卖铲子，你要找的是那些通过活动抽取费用、且活动量正在增长的业务。
1. “逃离朝九晚五”的产业体系巨大且仍在增长
2. 预测市场
3. 加密货币基础设施
4. 社交交易与社区
真正的赌注并非押注个别投机者获胜，而是押注这种现象将持续存在。押注驱使年轻人投身高风险赌局的根本经济环境不会改变。押注从这些活动中抽取佣金的平台将随用户规模持续扩张。
